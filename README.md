# Employee Management System (Spring Boot + MySQL)

A RESTful web application for managing employee records, developed using **Java**, **Spring Boot**, **Spring Data JPA**, and **MySQL**.  
This project demonstrates layered architecture, CRUD operations, and API testing with Postman.

| Update Employee | PUT | `/employees/update/{id}` | Updates existing employee |
| Delete Employee | DELETE | `/employees/delete/{id}` | Removes employee from database |
